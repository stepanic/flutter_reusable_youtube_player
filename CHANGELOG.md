# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1 - 2025-11-13

### Added
- Initial release of flutter_reusable_youtube_player
- Reusable YouTube player widget with customizable controls
- Player controller for managing playback state
- Support for various YouTube URL formats
- Video ID extraction and validation helpers
- Thumbnail URL generation utilities
- Duration formatting and parsing utilities
- Customizable player configuration options
- Custom player controls overlay
- Example application demonstrating all features
- Comprehensive test coverage
- Full documentation with usage examples
